公開實驗試玩沙盒私人分析

一份給人看的白話說明。這個網站不是報明牌，而是一台「證明或推翻樂透隨機性」的實驗機器。

這在做什麼？

我們同時跑 68 種選號策略（熱門號、冷門號、包牌、養牌、反熱門…）＋ 50 個純亂數對照組， 用完全相同的規則、超過 20 年的真實開獎資料回測，看有沒有任何一種策略「長期每一塊錢的報酬」 能顯著贏過亂數。

我們的預設預測：贏不了。如果連專家策略都贏不過丟骰子，那就用數據證明了「沒有明牌」。

誠實防自欺三件套：① 對照組（跟亂數比）② walk-forward（只能用過去、不能偷看未來） ③ 預先註冊（規則在看到結果前就凍結、公證上鏈）。

公開實驗頁怎麼看

  • 運氣雲：純亂數能有多好？淡灰是 50 個純亂數玩家的成績範圍，彩色是有邏輯的策略。只要沒有任何彩色線衝出灰色雲層，就代表「贏不了亂數」——目前正是如此。
  • 策略擂台：每張卡片一條選號規則，看它這期選了哪些號碼、為什麼、歷史每 100 元拿回多少。重點看有沒有誰贏過「最幸運的那個亂數玩家」（不是平均）。
  • 大家都買什麼號碼？哪幾期開出的號碼撞上大眾愛買的組合（紅點）。跟預測力無關，是觀察分獎人數的現象。
  • 搖獎機公平嗎？搖獎機／球是否公正（每顆球出現次數 vs 公平預期）。含 2004–2006 台北銀行時代的延伸比對（第三方資料，信任層級較低，不列入主結論）。
  • 實驗進度：前瞻段跑滿約 200 期後自動生成最終報告。現況：已預先註冊、等待前瞻段開跑

想試自己的號碼？到 試玩沙盒 用全歷史回測，看你的組合每 100 元能拿回多少、在 50 個亂數玩家中排第幾。（純沙盒，不進正式實驗。）

私人投注分析（你自己的彩券）

你每期實際買的號碼，可以在 私人分析頁 追蹤， 且號碼經加密、任何人（包含看原始碼的人）都看不到——密語只有你知道。

第一次設定 / 新增投注（全程在網頁，免終端機）

  1. 打開 私人分析頁，輸入你的通關密語（第一次用就輸入你想設的密語），按「解密」。
  2. 在「固定組」區塊，把夢境號碼／塔位號碼各設一次（名稱＋6 個號碼，空白分隔）。
  3. 每期買完，在「單期投注」填期別、花費，電腦選號填號碼，勾選要跟的固定組，按「新增這期投注」。上方表格會即時更新你的損益。
  4. 按 ⬇️ 下載 bets.enc。
  5. 到 GitHub 專案的 data/private/ 資料夾 → Add file → Upload files，把剛下載的 bets.enc 上傳覆蓋 → Commit。完成。

下次要看或編輯，回到私人分析頁輸入密語即可。密語與號碼永遠只在你的瀏覽器裡， 不會傳到任何伺服器；忘記密語＝資料救不回（這是刻意的安全設計）。

進階：也能用終端機

熟終端機的話，一樣可用 python -m tools.add_bet（init / add-recurring / add-bet / show）。 兩種方式產生的 bets.enc 格式相同、可互通。

真錢防火牆

系統只做虛擬損益。你實際下注多少、買什麼，是場外自己決定；私人投注紀錄純供你自己分析， 不會回饋進任何策略評分，不影響公開實驗的公正性。
