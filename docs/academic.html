§1 摘要

研究問題：是否有任何選號策略的長期理論軌每元回收，能顯著且穩健地贏過「同結構隨機下注」的對照分布？預設假說 預先註冊 的 H0＝不能。設計：68 條策略 ＋ 50 條純亂數對照，於 walk-forward 前視保護下對官方 2007+ 全歷史回測，置換檢定 ＋ BH-FDR 校正。

證據重心聲明：本實驗對隨機性的推論重心為硬體審計（對百分點級系統性偏差具實用檢定力）；策略競技場的科學角色為「預先註冊的公開示範」與「FDR 假陽性防火牆」，非隨機性結論的主要證據來源（詳 §3、§5）。

§2 方法

§2.1 預先註冊協定

啟動期 ；config 凍結 commit 之時間戳即為預先註冊時點（規格 §5.3）。策略清單、參數、統計方法、停止規則（前瞻段約 200 期）於啟動前凍結、公開於 repo，事後不改；發現 bug 循「修程式＋全量重跑＋公開留痕」路徑（§7）。本報告對應 commit 。

§2.2 資料來源與品質分層

主源：台灣彩券官方 JSON API（2007-01 起，期別自 96000001）。延伸層：2004–2006 北銀時代，兩個獨立第三方互相對帳（numbers_only、data_quality 較低），僅供硬體審計「延伸視圖」與分段檢定、永不進入策略推論。data_quality 三級（full／partial／numbers_only）依欄位完整度標記。信任分層理由：headline 結論站官方源；第三方資料受質疑時只傷延伸視圖、不傷主結論。

§2.3 Walk-forward 與前視保護

第 N 期選號僅餵入第 N 期以前的歷史；引擎層強制切片並設守門（含未來期的 history 丟 LookAheadError），即使策略自身不作弊亦保證不洩未來。測試套件含前視保護專項。

§2.4 雙軌 ROI 與加碼排除

推論主指標為理論軌每元回收：t1–t4 用開跑前凍結之長期典型獎金（中位數）、t5–t8 用固定獎額，隔離單期浮動頭獎與分獎人數污染，使策略間可公平比較。實際軌保留當期真實獎額，僅供展示、不進統計判斷。加碼獎項（節慶紅包）不計入任一軌（規格 §2.3、§4.2 凍結決策）。

§2.5 Permutation null 設計 ＋ BH-FDR

虛無分布以「同結構隨機下注」建構：固定真實開獎序列，將策略每期下注替換為同注數結構（single／7 碼包牌／8 碼包牌）的隨機注，重抽 n_perm 次得理論軌每元回收虛無分布；p =（1 + #{null ≥ 觀測}）/(n_perm + 1)。null 的隨機化維度是「下注」而非「開獎」——開獎序列固定為真實歷史，故不存在打散時序結構的問題。跨 N 條策略以 BH-FDR 校正，N ＝ 全部註冊策略數，於凍結時寫死。

§2.6 硬體審計

獨立於策略驗證，用全歷史：單號頻率、特別號、配對共現、分段（依年）卡方適合度檢定。明確不做 NIST 套件。Monte Carlo 校準（§7 裁決 A）：大樂透每期同時抽 6 顆不放回，49 顆球次數兩兩負相關，統計量 S=Σ(O−E)²/E 在虛無下 E[S]=49×(1−6/49)=43＜χ²₄₈ 期望 48，直接查表偏保守；改以模擬公平開獎建實證虛無分布計 p，並以解析縮放 p=P(χ²₄₈ ≥ S×48/43) 交叉驗證。特別號每期一顆、跨期獨立，χ²₄₈ 成立，不校準。標準化殘差 z=(O−E)/√(E(1−p))。混池限制：台彩定期輪換球組與搖獎機，長期混池檢定只能偵測長期系統性偏差。

§3 檢定力聲明

本段量化「接受 H0 ≠ 證明完美隨機」。MDE（最小可偵測效應量）採操作型定義（裁決書附錄 J/K）：模擬植入一條每元回收 +δ 的策略（各獎項命中率等比提升），對真實 null 分布計 permutation p 並經 BH-FDR 校正，δ₈₀ ＝ 偵測率達 80% 的最小 δ。口徑為完整理論軌（含頭獎），與註冊檢定一致。

§3.2 機制：為何門檻如此之高

200 期單注 null 的極端尾巴由離散跳階事件構成：一次三獎（t3）命中即使每元回收跳到 級。三獎每期命中率約 0.36%，高於 BH 最嚴門檻（0.05/N ≈ 0.074%），故決定顯著與否的臨界值被墊至「一次三獎」的水準。頭獎（t1，約 0.0014%/200 期）與貳獎（t2，約 0.0086%）低於門檻，落在完全盲區——即使把頭獎機率翻倍，偵測率仍約 。敏感度（附錄 K.2-6）：僅提升 t3–t8 的 δ₈₀ 仍屬同一數量級，帳面 δ 中約 流經 t1/t2 盲區——確認頭條數字非參數化假象。

§3.3 結論翻譯

本實驗前瞻段的「接受 H0」不構成「策略無效」的有力證據，僅表示：若任何策略存在真實優勢，其幅度小於本設計可偵測的門檻——而該門檻經模擬量化後極高（見 §3.1 δ₈₀ 表），高於一切現實中可信的選號優勢。因此策略競技場的科學角色是：①預先註冊的公開示範與誠實承諾機制；②FDR 控制下的假陽性防火牆。它不是本站隨機性結論的主要證據來源。

§5 限制

  1. 檢定力邊界（核心）：如 §3，前瞻段對現實中的微弱選號優勢檢定力趨近零。此邊界與 F 組「反熱門」策略的分獎通道盲區為同一結構——理論軌刻意用凍結獎額隔離分獎波動，因此「避開大眾號→中獎時少分人→期望值較高」這個真實存在的優勢，在理論軌回測分數上量測不到。反熱門優勢是理論軌設計下的量測盲區；本實驗以熱門度分析間接驗證其前提，不宣稱直接量測。
  2. F 組不可測性：同上，分獎人數優勢不在理論軌口徑內。
  3. 北銀資料信任層：2004–2006 為第三方雙源對帳、numbers_only，僅入延伸視圖，永不進策略推論。
  4. 球組輪換混池：長期混池卡方只能抓長期系統性偏差，抓不到單一球組短期物理瑕疵。
  5. 電腦選號稀釋：熱門度分析測到的是「剩餘人為偏好強度」，電腦選號（均勻隨機）會稀釋訊號。

§7.1 裁決 A：硬體審計卡方 Monte Carlo 校準（修保守偏誤）

不放回抽 6 顆使教科書 χ²₄₈ 系統性偏保守；改 MC 校準（單號＋配對）＋解析交叉驗證，特別號不校準。全量重跑前後 p 對照：

結論方向不變（全不顯著＝結果甲）。標準化殘差 z 併加 (1−p) 因子（單號 43/49、特別號 48/49）。

§7.2 裁決 J/K：檢定力 MDE 由 mean/SD 式改為操作型

初版以 mean/SD 位移模型計 MDE，在含凍結頭獎的重尾 null 下產生天文數字；根因為 mean/SD 描述法對秩次型檢定無效（臨界值由 null 極端百分位決定，不受頭獎稀有事件影響）。改操作型 δ₈₀（本 §3）。附錄 J 對數量級的預期為未經計算之錯誤估計，於此修正。詳見 DECISIONS.md。
